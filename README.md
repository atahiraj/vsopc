
## About this project

This project implements a compiler for the VSOP language. VSOP is a very simple object oriented programming language created at the University of Li√®ge. Specifications for the language may be found in `ressource/VSOP.pdf`.

## Getting started

Build with

```sh
$ make
```

The output is named `vsopc`

## Usage

Run
```sh
$ ./vsopc <options> file
```

See `ressource` folder fore more information on the compiler and the VSOP language.
